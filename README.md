# java-mini-projects

Repositório com mini-projetos em **Java** criados para aprimorar habilidades práticas e consolidar conceitos
fundamentais da linguagem.

## Sobre

Este repositório reúne exemplos pequenos e focados para estudar e demonstrar conceitos de programação orientada a
objetos (POO), herança, polimorfismo, tratamento de exceções e estruturas de dados básicas em Java. Cada mini-projeto é
intencionalmente simples — pensado para facilitar leitura, experimentação e evolução incremental.

## Conteúdo

Projetos atuais (cada pasta contém código-fonte com `Main` ou ponto de entrada equivalente):

**Projetos `projects/`:**

- **bank_account:** Simulação de contas bancárias com operações básicas
- **calculator:** Calculadora simples com operações aritméticas
- **contact_list:** Gerenciamento de lista de contatos
- **conversor:** Conversão entre celsius e fahrenheit
- **employee_management:** Sistema básico de gerenciamento de funcionários
- **library_system:** Gerenciamento de livros em uma biblioteca
- **person_register:** Registro simples de pessoas
- **student_manager:** Gerenciamento de estudantes e suas notas
- **todo_list:** Lista de tarefas com funcionalidades de adicionar, remover e listar

**Classes utilitárias e trechos de código `snippets/`:**

- **collections:** Exemplos de uso de listas, conjuntos e mapas (ArrayList, HashSet, HashMap) e operações comuns.
- **concurrency:** Padrões e utilitários para concorrência (Threads, Executors, sincronização, locks).
- **date:** Manipulação de datas e horas com `java.time` (LocalDate, LocalDateTime, DateTimeFormatter).
- **error_handling:** Tratamento de exceções, criação de exceções customizadas e boas práticas de try/catch/finally.
- **file:** Operações de I/O em arquivos e diretórios usando `java.nio.file` e streams de arquivo.
- **generics:** Exemplos de generics, wildcards (`? extends`, `? super`) e tipos parametrizados.
- **inner_classes:** Declaração e uso de classes internas, estáticas e anônimas, e quando preferir cada uma.
- **internationalization:** Exemplo de internacionalização de textos.
- **lambdas:** Sintaxe de expressões lambda, `FunctionalInterface` e uso com streams/collections.
- **optional:** Uso de `Optional` para evitar `null`, métodos úteis (`map`, `flatMap`, `orElse`).
- **path:** Manipulação de caminhos de arquivos com `Path`, `Paths` e operações de resolução/normalização.
- **regex:** Exemplos de expressões regulares, `Pattern` e `Matcher` para validação e extração.
- **streams:** Operações funcionais com `Stream` (map, filter, collect, reduce) e pipelines.
- **strings:** Manipulação de `String` e `StringBuilder`, formatação e métodos comuns.
- **threads:** Criação e controle de threads, ciclo de vida, comunicação entre threads (wait/notify).

> Observação: estes são exemplos práticos — cada pasta está isolada para facilitar execução e testes. Novos
> mini-projetos serão adicionados conforme avanço dos estudos.

## Boas práticas adotadas

- Código legível e organizado por responsabilidade (classes pequenas e focadas).
- Tratamento explícito de exceções quando necessário.
- Encapsulamento e uso de construtores para modelagem de objetos.
